Proyecto: Libro de recetas

Roles de los integrantes:
 ○ Base de datos: Martinez
 ○ Interfaz gráfica: Grilletti, Bastán 
 ○ Programación: Vitale


Objetivo del proyecto: El proyecto es un libro de recetas en el que uno podrá buscar las recetas segín los ingredientes que se disponen. Las comidas deben poder filtrarse segun su categoría (Postre, plato principal, bebida, panadería, etc.). En la página de cada receta, se deberá ver una valoración general y será posible escribir comentarios/reseñas sobre la misma. Naturalmente, es necesario disponer de una cómoda interfaz.

Backend: PHP
Frontend: HTML5, CSS3, JS
Database: MongoDB
